1. Besides 4x4, what other dimensions does the framework allow?
The game also supports 5x5, 6x6, 7x7, 8x8, and 9x9 board frameworks.

2. What data structure is used for representation of the game board?
A square multidimensional array is used for representation of the board.

3. What function is called to greet the player?
The function greet() is called at the start of the game.

4. What functions do you need to implement?
int(), draw(), move(), and won() are implemented.

_______________________________________________________________________________

To compile: $ make
To play: ./fifteen $ board_dimension
